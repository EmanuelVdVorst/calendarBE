# Calendar Backend

Node.js + Express + TypeScript backend API for the Calendar application.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Server will run on `http://localhost:5000`

## Build

```bash
npm run build
```

## Production

```bash
npm start
```

## Available Endpoints

- `GET /` - API status
- `GET /api/health` - Health check
